# Gelato Landing Page

Landing page for the Gelato ice cream delivery platform, inspired by goodlood.com.

## 🎨 Brand Colors

- **Espresso (Primary):** `#4a044e`
- **Amber (Secondary):** `#b45309`
- **Soft Canvas (Background):** `#fffdfa`

## 🚀 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **i18n:** next-i18next (PL, EN, UA)
- **Deployment:** Vercel (recommended)

## 📦 Installation

```bash
npm install
npm run dev
```

## 🌍 Multi-language Support

Supports: Polish (default), English, Ukrainian

Translation files: `/public/locales/{pl,en,ua}/common.json`
